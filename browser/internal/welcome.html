Site Modes

Stealth uses so-called Site Modes. Site Modes influence what type of content and media is requested from from that particular domain.

By default, Stealth will not request anything to ensure maximum control over your privacy and security.

Mode Behaviour
Allow/Disallow Texts
Allow/Disallow Images
Allow/Disallow Audios
Allow/Disallow Videos
Allow/Disallow Downloads

Keyboard Shortcuts

Keyboard Shortcuts are mapped in the graphical order in which the Widgets appear on the App Bar, from left to right, with exceptions to the shortcuts for Tab interaction.

Convenience shortcuts from other Browsers are also integrated, for people with a too strong muscle memory.

Key Behaviour
[Esc] Hide Context Menu, Unfocus Address, Hide Sidebars
3x [Esc] Show Help Overlay
[F1] Visit earlier Site
[F2] Visit later Site
[F3] Refresh/Pause current Tab
[F4] Open new Tab
[F5] Edit current Tab URL
[F6] Kill current Tab
[F7] Show previous Tab
[F8] Show next Tab
[F9] Toggle through Site Modes
[F10] (reserved)
[F11] Toggle through Beacon/Session/Site Settings
[F12] Open Browser Settings
[Shift]
+
[Tab]		 Navigate to previous Button, Input or Link
[Tab] Navigate to next Button, Input or Link
[Ctrl]
+
[Space]		 Toggle Context Menu
