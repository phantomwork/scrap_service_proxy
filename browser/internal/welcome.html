Site Modes

Stealth respects so-called Site Modes. These Site Modes influence what type of content and media is requested from from that particular domain.

By default, Stealth will not request anything to ensure maximum control over your privacy and security.

Mode Behaviour
Allow/Disallow to request Texts
Allow/Disallow to request Images
Allow/Disallow to request Audios
Allow/Disallow to request Videos
Allow/Disallow to request Downloads

Keyboard Shortcuts

Stealth has the keyboard shortcuts mapped in the graphical order the buttons appear on the header bar, from left to right; with exceptions to the shortcuts for tab control. Convenience shortcuts are also integrated, for people that come from other Web Browsers with a too strong muscle memory.

Shortcut Behaviour
[F1] Visit earlier Site
[F2] Visit later Site
[F3] Refresh/Pause current Tab
[F4] Open new Tab
[F5] Edit current Tab URL
[F6] Kill current Tab
[F7] Show previous Tab
[F8] Show next Tab
[F9] Toggle through Site Modes
[F10] (reserved)
[F11] Toggle through Beacon/Site/Peer Sidebars
[F12] Open Browser Settings
Shortcut Behaviour
[ESC] Hide Context Menu, Unfocus Address, Hide Sidebars
[Ctrl] + [W] Kill current Tab
[Ctrl] + [E] Edit current Tab URL
[Ctrl] + [R] Refresh/Pause current Tab
[Ctrl] + [T] Open new Tab
[Ctrl] + [Backspace] Toggle through Beacon/Site/Peer Sidebars
[Ctrl] + [Space] Toggle Context Menu
[Ctrl] + [Page Up] Show previous Tab
[Ctrl] + [Page Down] Show next Tab
