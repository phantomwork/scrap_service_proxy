Site Modes

Stealth respects so-called Site Modes. These Site Modes influence what type of content and media is requested from from that particular domain.

By default, Stealth will not request anything to ensure maximum control over your privacy and security.

Site Mode Request Behaviour
Toggles to request Texts from Peers or the Internet
Toggles to request Images from Peers or the Internet
Toggles to request Audios from Peers or the Internet
Toggles to request Videos from Peers or the Internet
Toggles to request Downloads from Peers or the Internet

Keyboard Shortcuts

Binding Behaviour
[ESC] Unfocus Address Header, Hide all Sidebars
[F1] Focus Address Header, ready to type
[F2] Rotate through Stealth Modes
[F3] Toggle Site Modes/Filters/Optimizers Sidebar
[F4] Open Stealth Browser Settings
[F5] Rotate through Tabs backwards
[F6] Rotate through Tabs forwards
[F7] Kill the first Tab
[F8] Kill the least Tab
[Ctrl]+[R] Reload current Tab
[Ctrl]+[ESC] Close current Tab
