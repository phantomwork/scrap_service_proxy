Internet

The Stealth Browser can connect to a website through direct internet access machines and peer-to-peer internet access machines.

If this machine does not have internet access, select the peer-to-peer option. It will get internet access once a peer that has internet access is reachable in the local network.

Deep Web

The Stealth Browser supports using a TOR Proxy to access the Deep Web.

If a TOR proxy is used, local peer discovery doesn't work anymore. This machine has to have mobile or broadband internet access to connect to sites on the Deep Web.

Hosts

The Stealth Browser has its own persistent DNS cache in order to prevent tracking capabilities of Internet Service Providers when being online via public Wi-Fi or 2G/3G/4G.

If both the current internet connection and the Domain support IPv6, the Stealth Browser will connect to it via IPv6.

It is possible to use transparent IPv6 proxies here, so for example when github.com only supports IPv4 and ikvjwd.com supports IPv6, you can manually edit the IPv6 of github.com and use the IP of the IPv6 proxy.

Peers

Trusted Peers allow the efficient sharing of cached resources among internal (or external) networks by using the WS/13 protocol.

The status indicates whether or not it was possible to connect to the peer, whereas the connection represents the type of internet access the peer has.

If you do not trust a specific peer anymore, you can remove it.

Domain Connection Status

Site Modes

Site Modes allow to request different media types by using a whitelist-based configuration. Please select the correct media types in the following table.

If you do not trust a specific domain anymore, you can remove its Site Mode. The Stealth Browser will not request anything by default.

Domain Site Mode

Site Filters

Site Filters allow to request specific URLs on a domain by using whitelist-based rule configurations. If a Site Mode allows a specific media type and a Site Filter for that domain is present but does not explicitely allow it, it will not be requested.

If you do not trust a specific domain anymore, you can remove its Site Filters and therefore the Site Mode (if present) will take over.
